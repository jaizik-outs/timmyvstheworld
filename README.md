# Timmy vs. the World

Timmy has big dreams of changing the world, one financed burrito at a time, but does he have what it takes to see it through? 
A first playable Vite + TypeScript + Phaser 3 prototype with one Pac-Man-style maze. 

## Run

🛠️ Step 1: The One-Time Setup (If you haven't done this before)
Download and install Node.js (this lets you run the server): https://nodejs.org/ (Just download the Recommended/LTS version).

Download and install Git (this lets you download the code): https://git-scm.com/downloads.

🚀 Step 2: Download and Run the Game
Open your computer's terminal (or Terminal on Mac / Command Prompt on Windows) and run these commands one line at a time:

Bash
# 1. Clone (download) the project onto your computer
git clone https://github.com/jaizik-outs/timmyvstheworld.git

# 2. Move into the project folder
cd timmyvstheworld

# 3. Install all the game dependencies/packages
npm install

# 4. Start the local server!
npm run dev
🕹️ Step 3: Play the Game
Once that last command finishes running, it will give you a local web link. Open your browser and go to:
👉 http://localhost:5173
